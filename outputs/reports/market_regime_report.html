1. What the Model Found

One OLS model per market regime: aFRR_Price = β₀ + β₁ · CSS where CSS = DA_Price − Gas_Price/η − ETS·intensity/η.

The CSS slope β₁ quantifies how sensitive aFRR bid prices are to plant profitability. Regime classification already captures CCGT generation levels, so CSS is the sole predictor.

A positive β₁ means higher CSS (more profitable spot sales) → higher aFRR bid prices. The Low regime has the strongest relationship because scarce reserve supply amplifies the effect of plant opportunity costs.
